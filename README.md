# 🏦 Banking System (Django - Python3)

Bank account system with complete CRUD operations for Savings/Current accounts

**Charles Ajeigbe's quote**:  “The technology you use impresses no one. The experience you create with it is everything.” – **Sean Gerety**


## 📌 Overview
This is a secure and scalable banking system built with Django and Django REST Framework.  
It supports user authentication, account management, and financial transactions.

---

## 🚀 Features
- User Registration & Authentication
- Account Creation & Management
- Deposit, Withdrawal, and Transfers
- Transaction History
- Admin Dashboard
- Secure Transaction Handling

---

## 🏗️ Tech Stack
- Python 3
- Django
- Django REST Framework
- PostgreSQL
